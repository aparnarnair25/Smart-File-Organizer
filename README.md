# Smart File Organizer

A Python automation tool that organizes files automatically into categories such as Images, Documents, Music, and Videos.

## Features

* Automatic file categorization
* Folder creation
* File movement
* Activity logging
* Duplicate file handling
* Progress tracking

## Technologies Used

* Python
* os module
* shutil module
* datetime module

## How to Run

```bash
python organizer.py
```
